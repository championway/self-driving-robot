# Installation

## Package

```
$ sudo apt-get install ros-kinetic-gazebo-ros-*
```
# Self Driving Robot

```
$ cd self-driving-robot/catkin_ws
$ source environment.sh
```
Note:
Do it everytime as you open new terminals
